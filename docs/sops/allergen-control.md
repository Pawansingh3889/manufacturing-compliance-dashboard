# Allergen Control SOP

## Purpose
Prevent unintended cross-contact with any of the fourteen allergens
regulated in the UK and EU, and keep product labelling accurate.

## Scope
All production lines, storage areas, utensils and personnel handling
raw materials or finished product.

## References
- BRCGS Food Safety Issue 9, clause 5.3
- Food Information Regulations 2014
- UK Food Standards Agency allergen guidance

## Controlled Allergens
Fish, crustaceans, molluscs, milk, eggs, gluten, soya, sulphites,
mustard, celery, sesame, lupin, tree nuts, peanuts.

## Controls
- Allergen status is recorded on every product record and checked
  before each run.
- Dedicated utensils or documented cleaning validation between
  allergen and non-allergen runs.
- Rework of allergen-containing product is only permitted back into
  a run that already declares the same allergen.
- Label verification is performed at the start and end of every
  production run and recorded in the despatch record.

## Cross-Contact Incident Handling
If cross-contact is suspected:
1. Stop the affected line and quarantine product.
2. Investigate using the allergen matrix and production linkage.
3. If confirmed, relabel or dispose of product. Under no circumstances
   is product released for consumer sale with incorrect allergen
   declaration.
4. Trigger the recall procedure if product has already left the site.

## Compliance Target
Zero allergen labelling deviations per rolling twelve month window.
Any deviation is a major non-conformance for the purposes of the
annual BRCGS audit.
